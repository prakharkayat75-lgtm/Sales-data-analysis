 Sales Data Analysis Project

1 Overview
 
This project focuses on end-to-end data analysis using real-world sales data. The objective was to clean, transform, analyze, and visualize the data to extract meaningful business insights.

2 Tools & Technologies Used

* Microsoft Excel (Data Cleaning)
* PostgreSQL (Data Validation & Querying)
* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Power BI (Dashboard & Visualization)


3 Dataset Description

The project includes three main datasets:

* **Customers Data** (customer details like name, age, city)
* **Orders Data** (transaction details like order_id, qty, selling_price, date)
* **Products Data** (product category, brand, cost_price)


4 Data Cleaning Process

* Handled missing values in key columns (qty, prices)
* Fixed invalid date formats
* Removed duplicate records
* Standardized text fields
* Created new features: **Revenue & Profit**


5 Key Analysis Performed

* Monthly Sales Trend
* Category-wise Revenue Analysis
* City-wise Sales Distribution
* Top Customer Identification
* Profitability Analysis
* Year-over-Year (YoY) Growth


6 Dashboard Features (Power BI)

* KPI Cards (Revenue, Profit, Orders, Customers)
* Interactive filters (City, Category, Date)
* Trend analysis using line charts
* Customer and category insights


7 Key Insights

* Revenue is driven by a few top-performing categories
* Sales are concentrated in major cities
* Seasonal trends observed in monthly sales
* High-value customers contribute significantly to revenue
* Some transactions show negative profit

 Conclusion
This project demonstrates a complete data analytics workflow — from raw data cleaning to business insights and dashboard creation using Power BI.
